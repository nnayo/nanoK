NanoK
-----
AVR library with basic functionnalities and drivers:
 - timers, 
 - asynchronous I2C(twi), 
 - asynchronous SPI,
 - asynchronous UART,
 - asynchronous eeprom read/write

 - fifo, 
 - proto threads (pt)

 - sdCard read/write
 - accelerometer (adxl345) read / configure
 - ethernet chip W5100

Compilation:
------------
Use the scons program (http://www.scons.org/)
Simply run: 
  scons
